> ‚ö†Ô∏è Note: Some details on this page may not be fully up to date.

### Step 2 ‚Äî Download Multisim 14.3 Directly

Go to the official NI Multisim download page:

[https://www.ni.com/en/support/downloads/software-products/download.multisim.html#452133](https://www.ni.com/en/support/downloads/software-products/download.multisim.html#452133)

* Under **Included Editions**, select **Education**
* Then click **Download**
* Installation may take some time

### Step 3 ‚Äî Request a Student License

Go to the UW Licensing Portal:

[https://software.engr.uw.edu/software/labview](https://software.engr.uw.edu/software/labview)

1. Log in using your UW NetID
2. Select **LabVIEW ‚Üí LabVIEW ‚Äì Student Edition** (this includes Multisim licensing)
3. You will receive an email containing your license code

After installation, activate the license:

* Choose **Connect to a Volume License Server**
* Click **Connect**

Your Multisim license should now be active.
